Sales Data Analysis Project
Objective

Analyze the sales dataset to identify key revenue drivers and generate actionable insights.

Tools Used: 
Python
Pandas
Matplotlib

Key Insights

Top-performing product: Laptop
Regional performance: West region shows strongest sales
Average order value: ₹31,700
Total revenue: ₹3,17,000

Recommendations

Focus marketing and inventory efforts on high-performing product-region combinations.
Optimize sales strategy based on revenue-driving products and regions.

Conclusion
Data-driven insights can help the company make informed decisions for marketing, inventory management, and revenue optimization.
